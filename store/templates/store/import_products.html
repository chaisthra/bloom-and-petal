CSV Format Instructions

Your CSV file should have the following columns:

  • name (required): The product name
  • description: Product description
  • price (required): Product price (numeric value)
  • category (required): Category name (will be created if it doesn't exist)
  • stock: Available stock (integer)
  • available: Whether the product is available (true/false, yes/no, 1/0)

Example:

name,description,price,category,stock,available
Artificial Red Rose,"Beautiful red rose, looks real",19.99,Artificial Roses,50,true
Sunflower Bouquet,"Bright yellow sunflowers",29.99,Artificial Sunflowers,25,true
